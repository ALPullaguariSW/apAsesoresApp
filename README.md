# A&P Asesores App

<img src="./rLogo.svg" alt="Alt Text" style="width: 50%;">
Esta es una aplicación web para A&P Asesores.

## Descripción

La aplicación web proporciona funcionalidades para cotizar seguros, administrar pólizas y obtener información sobre los servicios de A&P Asesores.

# FRONTEND

## Instrucciones de Ejecución

1. 🚀 Clona este repositorio en tu máquina local:

```bash
   git clone <>
```

2. 📁 Accede al directorio del proyecto:

```bash
cd frontend
```

3. 💻 Instala las dependencias necesarias utilizando npm:

```bash
npm install
```

4. ▶️ Inicia la aplicación en modo de desarrollo:

```bash
npm run dev
```

5. 🌐 Abre tu navegador web y accede a http://localhost:3000 para ver la aplicación en funcionamiento.

## Dependencias

- React.js
- React Router
- Chart.js
- Tailwind CSS

